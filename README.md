# Aparna Tiwari Portfolio

Static portfolio website for Aparna Tiwari.

## Files

- `index.html` - portfolio content
- `styles.css` - responsive design
- `script.js` - mobile menu interaction
- `assets/` - add portfolio images here

## Image Names

Save your uploaded images with these names to make them appear automatically:

- `assets/hero.jpg`
- `assets/team.jpg`
- `assets/economic-times.jpg`
- `assets/exora.jpg`
- `assets/certificate.jpg`
- `assets/hackathon.jpg`
- `assets/group.jpg`

## Run Locally

Open `index.html` in a browser.

## Deploy

Upload the `aparna-portfolio` folder to GitHub and import it into Vercel as a static site.
